# Read ME
## Local Up

1. make local-up
```
build dist
start web server https://scaleo-client.loc:8888/
```
2. make local-down
```
stop web server
```
3. make local-build
```
rebuild angular docker image
```
4. make local-build-dist
```
rebuild dist
```
5. make local-serve
```
start debug web server http://localhost:9999/
```
